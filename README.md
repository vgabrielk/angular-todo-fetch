# Todo angular

Esse projeto tem como objetivo melhorar meu aprendizado no Framework Angular e absorver alguns conceitos.

## Servidor de desenvolimento

#### Rode os comandos abaixo para rodar o projeto localmente
##### Para instalar as dependências
```
npm install
```
##### Após instalar as dependências, rode o projeto com o comando
```
ng serve
```
#### O servidor irá iniciar na porta `http://localhost:4200/`
